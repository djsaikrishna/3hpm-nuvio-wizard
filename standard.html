Module 01 · Connected Services

The multi-addon manifest wizard for Nuvio.

Build one or more clean, keyless manifests for Comet, Torrentio, and StremThru Torz. Connect TorBox in Nuvio first, make sure Resolve playable links is enabled, then generate the addon URLs. No TorBox API key goes into this wizard.

AIOStreams + Nuvio Instant.

Import the tested 3HPM template, skip unnecessary debrid setup, add your TMDB credentials, and create the finished AIOStreams addon.

Step 01

Open AIOStreams

Go to AIOStreams and scroll to the Get Started section. Tap Use a Template. You do not need to start a full manual setup.

Step 02

Open the import option

In the template list, tap the download/import button in the bottom-right corner. This opens the template import screen where you can bring in your local template file.

Step 03

Import the local template file

Tap Import from File and choose the AIO with Nuvio Instant template JSON file from your device. You can also paste a template URL if you are using a hosted link.

https://nuvioinstant.3hpm.ca/Aiotb/aio-with-nuvio-instant-template.json
Step 04

Confirm the import

Review the template details. You should see AIO with Nuvio Instant, author 3HPM, and category Debrid. Tap Use This Template Now.

Step 05 · Do not miss this

Skip services

When AIOStreams asks you to select services, tap Skip. For this Nuvio Instant setup, Nuvio handles TorBox through its own Connected Services integration, so TorBox does not need to be configured inside AIOStreams.

Step 06

Enter TMDB credentials

Enter your TMDB API Key and TMDB Access Token. These are personal credentials, so they are not included in the template. After entering them, tap Load Template.

Need TMDB credentials?

Create a free TMDB account, generate an API Key v3 and Read Access Token v4, then return to AIOStreams and paste them into the fields shown.

Open TMDB Documentation ↗
Step 07

Create and install

Create a password for your AIOStreams configuration. This password protects the config so you can update it later. After creating it, continue through AIOStreams to install the finished addon.

Finished

The template does the heavy lifting.

Before installing, review the imported settings. The only user-specific pieces are your TMDB credentials and configuration password.

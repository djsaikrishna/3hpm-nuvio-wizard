How to use it

  1. In Nuvio, connect TorBox under Settings → Connected Services → TorBox.
  2. Pick a preset here. Do not enter a TorBox API key anywhere on this page.
  3. Copy the generated manifest URL and add it to Nuvio.
  4. Rename the addon in the Nuvio web account to the suggested 3HPM name.

This wizard is intentionally narrow. It exists to make Nuvio Connected Services setup less confusing, not to replace Comet’s full configure page.

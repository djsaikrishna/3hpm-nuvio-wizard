How to use it

  1. Connect TorBox in Nuvio: Settings → Connected Services → TorBox.
  2. Pick a preset here and copy the generated manifest URL.
  3. Add the manifest URL in Nuvio.
  4. Rename the addon in the Nuvio web account to the suggested 3HPM name.

No TorBox API key belongs in this wizard. If Nuvio changes Connected Services behavior, this tool may need adjustment.
